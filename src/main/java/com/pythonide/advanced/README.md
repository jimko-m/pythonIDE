GitIntegration.java - تكامل نظام Git\n**الغرض**: إدارة نظام التحكم في الإصدارات والـ commit history\n\n**الميزات الرئيسية**:\n- كشف التغييرات التلقائي\n- إدارة الفروع (Branches)\n- تاريخ الـ commits مع التفاصيل\n- إشعارات فورية عند التغييرات\n- دعم العمليات المتوازية\n\n**أمثلة الاستخدام**:\n```java\nGitIntegration git = new GitIntegration(\"/path/to/project\");\n\n// كشف التغييرات\nList<GitChange> changes = git.detectChanges().get();\n\n// إنشاء commit\nString commitHash = git.commitChanges(\"إضافة ميزة جديدة\").get();\n\n// إنشاء فرع جديد\ngit.createBranch(\"feature/new-feature\");\n```\n\n### 2. KeyboardShortcuts.java - اختصارات لوحة المفاتيح\n**الغرض**: إدارة شاملة للاختصارات والسياق\n\n**الميزات**:\n- اختصارات قابلة للتخصيص\n- سياقات مختلفة (Editor, Terminal, Debugger)\n- تنفيذ الأوامر الذكي\n- اكتشاف تضارب الاختصارات\n- إشعارات بصرية للصم وضعاف البصر\n\n**أمثلة الاستخدام**:\n```java\nKeyboardShortcuts shortcuts = new KeyboardShortcuts();\n\n// تسجيل اختصار جديد\nshortcuts.registerCommand(\"custom.action\", \"إجراء مخصص\",\n    new ShortcutKey(\"F12\", false, false, false, \"اختبار مخصص\"),\n    () -> performCustomAction(), Context.EDITOR);\n\n// تغيير السياق\nshortcuts.setCurrentContext(Context.DEBUGGER);\n\n// معالجة مفتاح مضغوط\nboolean handled = shortcuts.handleKeyPress(\"F5\", false, false, false);\n```\n\n**الاختصارات الافتراضية**:\n- `Ctrl+N`: ملف جديد\n- `Ctrl+S`: حفظ\n- `Ctrl+F`: بحث\n- `Ctrl+H`: بحث واستبدال\n- `F5`: بدء التشخيص\n- `Shift+Enter`: تشغيل الخلية (Data Science)\n\n### 4.
